# {cotoss}
CSS upside down preprocesor that compiles into CSS... or not.

Cotoss offers you a brand new ways to create unsense code like:

```css
.yourclass {
   display: 'maybe';
}
```

```css
.yourclass {
   position: 'over there';
}
```

```css
.yourclass {
   color: 'dark blue almost black';
}
```
